# @openacme/llm-provider

LLM provider abstraction for **OpenAcme**. Wraps the Vercel AI SDK to give a single interface across OpenAI, Anthropic, Google, and any OpenAI-compatible endpoint.

## Install

```sh
npm install @openacme/llm-provider
```
