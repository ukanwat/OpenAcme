# @openacme/db

SQLite-backed persistence layer for **OpenAcme**, using [drizzle-orm](https://orm.drizzle.team) over [better-sqlite3](https://github.com/WiseLibs/better-sqlite3).

## Install

```sh
npm install @openacme/db
```
