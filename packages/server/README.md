# @openacme/server

HTTP server for the **OpenAcme** AI agent platform. Built on [Hono](https://hono.dev). Most users want [`@openacme/cli`](https://www.npmjs.com/package/@openacme/cli) — this package is the underlying server it runs.

## Install

```sh
npm install @openacme/server
```
