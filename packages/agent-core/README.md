# @openacme/agent-core

Agent runtime for the **OpenAcme** AI agent platform. Exposes the `Agent` class, a system-prompt builder, and the streaming types used across the platform.

## Install

```sh
npm install @openacme/agent-core
```

## Usage

```ts
import { Agent } from "@openacme/agent-core";
```
