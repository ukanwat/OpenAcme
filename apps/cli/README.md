# @openacme/cli

The CLI for the **OpenAcme** AI agent platform. Installs an `openacme` binary that runs the agent server and a chat client.

## Install

```sh
npm install -g @openacme/cli
```

## Usage

```sh
openacme setup        # interactive first-time setup
openacme              # start the server
openacme chat         # start an interactive chat session
```

Run `openacme --help` for the full command list.
